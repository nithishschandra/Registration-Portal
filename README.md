>> Event Horizon – Registration Portal

This project is a web-based registration system created to manage event sign-ups in a simple and efficient way. It focuses on providing a clean interface for users and a reliable backend for handling authentication and data storage.

>> Technologies Used

Frontend: HTML and CSS with custom font styling using Cinzel

Backend: Firebase Authentication and Realtime Database

Hosting: Firebase Hosting platform

>> Main Features

User Authentication: Login and signup system implemented using Firebase with proper validation and error messages

Responsive Layout: Designed to work smoothly on both mobile phones and desktop screens using Flexbox and Grid

Interactive Design: Styled buttons, smooth animations, and transitions to improve user experience

>> Project Files

index.html – Main page for login and access

signup.html – Registration page for new users

auth.js – Firebase setup and authentication handling

style.css – Styling and layout customization
